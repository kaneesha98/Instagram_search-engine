# Instagram_search-engine
A simple CRUD app that interacts with a database. Different search options are:
1. Client can search the Pictures with the hashtag as a keyword for search.
2. Client can search the related pictures with the help of the pictures (search the picture through picture)
3. A client can provide the Account name as keyword for search.
An advance option is also available to filter the Pictures through Account, dates of creation, other hashtags or some keywords.
